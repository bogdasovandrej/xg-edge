# xg-edge live site

Responsive public dashboard for the result-free forecasts generated by the
parent `xg-edge` project. The browser refreshes the versioned
`reports/live_predictions.json` snapshot from GitHub every five minutes.

The UI exposes uncertainty, separate model/competition CLV cohorts and exact
timestamped bookmaker snapshots. Stale, post-kickoff or pre-forecast prices are
hidden. A forecast is not presented as a betting recommendation; the global
gate remains `NO BET`, while every cohort has its own fixed-horizon audit.

`npm test` builds and server-renders the page. `npm run export:static` creates
the GitHub Pages artifact under ignored `out-static/`.
