# Реестр метрик

Больше метрик ≠ лучше модель: каждый признак — степень свободы, на которой
модель может выучить шум. Метрика попадает в модель только через процедуру
допуска ([docs/hypotheses.md](hypotheses.md)), провал документируется.

## Ярус 1 — ядро (высокий сигнал)

| Метрика | Статус | Где в коде |
|---|---|---|
| xG за/против, скользящее окно с time-decay | ✅ в модели (H8 подтверждена) | `features/builder.py` |
| Домашний/выездной сплит рейтингов (venue blend 0.3) | ✅ в модели | `features/builder.py` |
| Полураспад веса матча (180 дней) | ✅ в модели, подобран на валидации | `features/builder.py` |
| Понижение веса матчей с удалениями (×0.5) | ✅ в модели; минутные мультипликаторы — после event-данных (H3) | `features/builder.py` |
| npxG вместо xG | ❌ H7 провалена — вклада нет, флаг `use_npxg` сохранён для перепроверки | `features/builder.py` |
| Нормировка на силу соперника (непрерывный коэффициент, клип [0.5, 2.0]) | ❌ H2 провалена — вклада нет, флаг `adjust_opponent` сохранён | `features/builder.py` |
| Домашнее преимущество как оцениваемый параметр | ✅ в модели (коэффициент GLM / параметр DC) | `models/` |

Активных признаков в GLM: 3 (атака, оборона соперника, домашний флаг) —
бюджет сложности (≤ 8–12) соблюдён с запасом.

## Ярус 2 — обогащение (правдоподобный сигнал, требует допуска)

Данные уже собираются в cleaned-слое, в модель не входят:

| Метрика | Данные | Гипотеза |
|---|---|---|
| PPDA (интенсивность прессинга) | ✅ есть (Understat) | H18 → рынки тоталов |
| Deep completions (территориальное доминирование) | ✅ есть (Understat) | H17 |
| xG per shot (качество момента) | требует shot-level данных | H12 |
| PSxG − GA (качество вратаря) | требует FBref | H14 |
| xG со стандартов отдельно от игры с хода | требует event-данных | H13 |
| Дни отдыха / плотность календаря | вычислимо из cleaned | H15 |
| Стабильность состава, травмы ключевых игроков | требует Transfermarkt/API-Football | H16 |

## Ярус 3 — экспериментальные (шумные)

Судейский профиль (карточки/пенальти), движение линии открытие→закрытие (H20,
отдельная ветка), погода, перелёты, game-state-скорректированный xG (H19),
momentum-метрики (только live-ветка).

## Явно исключено (мусорные метрики)

- Владение мячом как самостоятельный признак — заменяется field tilt / deep
  completions.
- «Мотивация», «характер», «класс» без параметризации.
- Серии («выиграли 5 подряд») — информация уже в xG с time-decay.
- Личные встречи глубже 2 лет — составы другие, информация мертва.
- Товарищеские матчи — исключены на уровне данных.
